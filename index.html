**Como anexar seus E-books no GitHub:** 1. No seu repositório `guia-cuidador`, crie uma nova pasta chamada `assets`. 2. Dentro de `assets`, crie outra pasta chamada `ebooks`. 3. Faça o upload dos seus arquivos PDF (ex: `guia_pnc.pdf`, `direitos_idoso.pdf`, `modelos_contrato.pdf`) para essa pasta `ebooks`. 4. Os links deste arquivo `ebooks_list.html` já estão configurados para esses caminhos.
